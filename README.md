# Snake game using Python's turtle module

Short repo for a 2-hour class that introduces Python basics to highschoolers through a simple Snake game.

## Usage

Open the notebook and run the cells to play the game. Start by playing, then explore how the code works by tweaking cells and trying ideas. Errors are expected and part of the learning.

*Footnote:* The game source was based on https://gist.github.com/wynand1004/ec105fd2f457b10d971c09586ec44900 and rewritten.
